# Телефонный справочник

Веб-приложение на Django для централизованного хранения контактных данных сотрудников и внешних контрагентов компании.

[Демо-версия проекта](http://2.26.17.182:8080/)

## Возможности

- **Справочник сотрудников** — табличный список с пагинацией, карточка сотрудника с полной информацией
- **Внешние контакты** — категоризация по типу (клиент / партнёр / поставщик / другое)
- **Избранное** — добавление сотрудников и контактов в избранное
- **Организационная структура** — управление отделами и подразделениями
- **Поиск и фильтрация** — по имени, должности, отделу, подразделению, организации, категории
- **Аутентификация** — регистрация с подтверждением по email, вход/выход, смена пароля
- **Роли** — обычный пользователь и администратор с разграничением прав доступа

## Стек технологий

- **Backend:** Python 3, Django
- **База данных:** SQLite
- **Frontend:** HTML, CSS (кастомные стили), JavaScript
- **Email:** SMTP (Gmail) для подтверждения регистрации

## Установка и запуск

### 1. Клонировать репозиторий

```bash
git clone <https://github.com/mikkixx/ContactBook>
cd ContactBook/metanit
```

### 2. Создать и активировать виртуальное окружение

```bash
python -m venv venv

# Windows
venv\Scripts\activate

# Linux / macOS
source venv/bin/activate
```

### 3. Установить зависимости

```bash
pip install django
```

### 4. Настроить переменные окружения

В файле `metanit/settings.py` задать значения для email-рассылки (или вынести в `.env`):

```python
EMAIL_HOST_USER = 'your-email@gmail.com'
EMAIL_HOST_PASSWORD = 'your-app-password'
```

### 5. Применить миграции

```bash
python manage.py migrate
```

### 6. Создать суперпользователя (опционально)

```bash
python manage.py createsuperuser
```

### 7. Запустить сервер разработки

```bash
python manage.py runserver
```

Приложение будет доступно по адресу: [http://127.0.0.1:8000/](http://127.0.0.1:8000/)

## Основные URL-маршруты

| URL | Описание |
|---|---|
| `/` | Список сотрудников |
| `/employee/<id>/` | Карточка сотрудника |
| `/contacts/` | Список внешних контактов |
| `/contacts/<id>/` | Карточка внешнего контакта |
| `/favorites/` | Избранное |
| `/profile/` | Личный кабинет |
| `/structure/` | Организационная структура |
| `/login/` | Вход |
| `/register/` | Регистрация |

## Модели данных

- **Department** — отдел (уникальное название)
- **Subdivision** — подразделение (принадлежит отделу)
- **Employee** — сотрудник: ФИО, телефон, должность, этаж, кабинет, отдел, подразделение, роль, мягкое удаление
- **Contact** — внешний контакт: ФИО, телефон, email, организация, должность, категория, владелец
- **Favorite** — избранный сотрудник (пользователь → сотрудник)
- **ContactFavorite** — избранный контакт (пользователь → контакт)
